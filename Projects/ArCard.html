About This Project

Technology Used:
• Unity Engine
• Vuforia (for AR tracking)

Features:
• AR marker detection to spawn 3D objects
• Smooth animation and interaction with the AR model
• Clean and modern UI layout

Target Audience:
• Students exploring AR basics
• Educators needing simple AR demos
• Anyone interested in interactive 3D experiences

Challenges Faced:
• Adjusting object scaling so it fits the real-world marker
• Lighting differences causing unstable AR tracking
• Making the UI clean and easy to understand
